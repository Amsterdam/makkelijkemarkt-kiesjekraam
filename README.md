# Kies je kraam backend
### Uitvoeren met Node.js webserver (alleen voor development)

Draai `npm install`.

Maak een bestand genaamd `.env`, begin met de inhoud van `example.env` als basis. Vul de Daalder/Keycloak variabelen in; `.env` blijft lokaal en wordt niet gecommit.

### Compiling the SCSS
Om de SCSS naar CSS te compilen draai je `npm run sass`. Als je de wijzigingen wilt zien refresh je je browser.

### Compiling JS
You can simply run the command 'webpack'
